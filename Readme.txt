El siguiente codigo presenta una estructura de Cola Circular con las siguientes funciones.
Todos estos metodos recien un puntero Cola.
-crearCola:  instancia valores.
-enQueue: y valor entero, agrega dicho valor a la cola
-deQueue:  y devuelve el valor en la posicion frente, tambien lo remueve de la cola
-borrarCola:  y libera el espacio reservado en memoria.
-colaVacia: y devuelve True si esta vacia
-colaLLena: y devuelve True si esta llena
